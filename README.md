# BiliSpeed - B站倍速增强工具

一个浏览器扩展，为B站（哔哩哔哩）提供视频倍速自定义功能。

## 功能特性

### 1. 视频倍速自定义
- 添加自定义倍速值（0.1-10.0，步进0.1）
- 自定义倍速插入到原生倍速菜单中
- 支持记住上次选择的倍速值
- 支持添加/删除自定义倍速
- 自定义倍速与原生倍速智能合并，避免重复

### 2. 键盘快捷键增强
- **长按右箭头键（→）加速**：长按右箭头键可临时加速到指定倍速（默认3.0x），松开后恢复原速
- **A键快速切换倍速**：按A键可在当前倍速和设定倍速之间快速切换
- **分离A键倍速设置**：可设置A键使用独立的倍速值，与右箭头长按倍速不同
- **智能倍速记忆**：记住切换前的倍速值，实现精准恢复

### 3. 设置页面
- 所有设置集中在扩展选项页
- 无弹出窗口，保持界面简洁
- 使用Chrome存储同步设置
- 实时设置生效，无需刷新页面

## 文件结构

```
BiliSpeed/
├── manifest.json              # 扩展清单文件 (Manifest V3)
├── options.html              # 设置页面
├── options.js                # 设置页面逻辑
├── background.js             # 后台脚本（存储管理和图标点击事件）
├── content-video.js          # 视频页面增强脚本（主要功能实现）
├── content-live.js           # 直播页面脚本（保留文件，功能已移除）
├── icons/
│   ├── icon16.png            # 扩展图标 16x16
│   ├── icon48.png            # 扩展图标 48x48
│   └── icon128.png           # 扩展图标 128x128
└── README.md                 # 说明文档
```

### 文件说明

1. **manifest.json** - 扩展配置文件，定义名称、版本、权限、内容脚本等
2. **options.html/options.js** - 设置页面，提供自定义倍速和键盘快捷键配置界面
3. **background.js** - 后台脚本，管理设置存储和处理工具栏图标点击事件
4. **content-video.js** - 核心功能脚本，在B站视频页面注入，实现倍速自定义和键盘快捷键功能
5. **content-live.js** - 直播页面脚本（保留文件，当前版本无直播增强功能）
6. **icons/** - 扩展图标文件夹，包含16x16、48x48、128x128三种尺寸的图标

## 安装和使用

你可以通过以下两种方式安装本扩展，任选其一即可。

### 从商店安装
扩展已上架 Chrome 网上应用商店 和 Microsoft Edge 加载项，安装后会自动保持更新。

**Chrome 用户**  
1. 打开 [Chrome 网上应用商店](https://chrome.google.com/webstore/category/extensions)  
2. 搜索 **BiliSpeed**
3. 点击 **添加至 Chrome**  
4. 在弹出的权限确认框中点击 **添加扩展程序**  
5. 安装完成后，地址栏右侧会出现扩展图标

**Edge 用户**  
1. 打开 [Microsoft Edge 商店](https://microsoftedge.microsoft.com/addons)  
2. 搜索 **BiliSpeed**  
3. 点击 **获取** → **添加扩展**  
4. 安装完成后，扩展图标会出现在工具栏中

---

### 手动安装（无需商店）
如果你更倾向于手动安装（例如用于调试），可以按照以下步骤加载扩展。

### 1. 下载扩展包
- 打开本仓库的 [Releases](https://github.com/xia0x1/BiliSpeed/releases) 页面。
- 找到最新的版本（例如 `v1.0.0`），在 **Assets** 区域点击下载 `.zip` 文件（如 `BiliSpeed.zip`）。
- 解压到本地任意目录。

**2. 打开浏览器的扩展管理页面**  
- **Chrome**：在地址栏输入 `chrome://extensions` 并回车。  
- **Edge**：在地址栏输入 `edge://extensions` 并回车。

**3. 开启“开发者模式”**  
- 在页面右上角找到 **开发者模式** 开关，将其 **打开**。

**4. 加载扩展**  
- 点击左上角的 **加载已解压的扩展程序** 按钮。  
- 在弹出的文件夹选择窗口中，选中你刚才解压（或克隆）的 `BiliSpeed` 文件夹，然后点击 **选择文件夹**。

**5. 确认安装成功**  
- 扩展列表中会出现 `BiliSpeed` 图标，表示安装成功。  
- 首次安装后，建议 **刷新** 已打开的 B站 页面，以便扩展正常注入。

⚠️ **注意事项**  
- 每次启动浏览器时，可能会弹出“请停用开发者模式扩展程序”的提示，点击 **取消** 即可，不影响使用。  
- Release 版本不会自动更新，如需更新请重复上述步骤下载新版本并重新加载。

### 使用设置页面
1. 点击浏览器工具栏中的BiliSpeed图标
2. 或右键点击图标，选择"选项"
3. 在设置页面中配置：
   - 添加/删除自定义倍速值
   - 设置长按右箭头键的加速倍速值（默认3.0x）
   - 启用/禁用A键切换倍速功能
   - 配置分离A键倍速值（可为A键设置独立的倍速值）

### 键盘快捷键使用
- **长按右箭头键（→）**：长按可临时加速到设定倍速，松开恢复原速
- **按A键**：在视频页面按下A键，可在当前倍速和设定倍速之间快速切换
- **注意**：快捷键仅在B站视频页面有效，需要视频获得焦点

## 技术实现

### 视频倍速功能
- 使用MutationObserver监听DOM变化
- 检测 `.bpx-player-ctrl-playbackrate` 倍速按钮
- 在原生倍速菜单中插入自定义选项
- 通过修改 `video.playbackRate` 改变播放速度
- 使用 `chrome.storage.sync` 保存设置


### 设置同步
- 使用Chrome存储API同步设置
- 默认设置：自定义倍速列表为空
- 实时监听设置变化，即时生效

## 兼容性
- Chrome 88+ (支持Manifest V3)
- Edge 88+ (基于Chromium)
- 支持B站视频页
- 适配B站新版播放器界面

## 注意事项
1. 首次安装后需要刷新已打开的B站页面
2. 自定义倍速值与原生倍速重复时不会重复添加
3. 所有设置保存在浏览器本地，卸载扩展会清除
