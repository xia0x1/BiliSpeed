# BiliSpeed - B站倍速增强工具

一个浏览器扩展，为B站（哔哩哔哩）提供视频倍速自定义功能。

## 功能特性

### 1. 视频倍速自定义
- 添加自定义倍速值（0.1-10.0，步进0.1）
- 自定义倍速插入到原生倍速菜单中
- 支持记住上次选择的倍速值
- 支持添加/删除自定义倍速


### 2. 设置页面
- 所有设置集中在扩展选项页
- 无弹出窗口（popup），保持界面简洁
- 视频页不出现额外设置按钮
- 使用Chrome存储同步设置

## 安装和使用


### 使用设置页面
1. 点击浏览器工具栏中的BiliSpeed图标
2. 或右键点击图标，选择"选项"
3. 在设置页面中配置：
   - 添加/删除自定义倍速值

## 技术实现

### 视频倍速功能
- 使用MutationObserver监听DOM变化
- 检测 `.bpx-player-ctrl-playbackrate` 倍速按钮
- 在原生倍速菜单中插入自定义选项
- 通过修改 `video.playbackRate` 改变播放速度
- 使用 `chrome.storage.sync` 保存设置


### 设置同步
- 使用Chrome存储API同步设置
- 默认设置：自定义倍速列表为空
- 实时监听设置变化，即时生效

## 兼容性
- Chrome 88+ (支持Manifest V3)
- Edge 88+ (基于Chromium)
- 支持B站视频页
- 适配B站新版播放器界面

## 注意事项
1. 首次安装后需要刷新已打开的B站页面
2. 自定义倍速值与原生倍速重复时不会重复添加
3. 所有设置保存在浏览器本地，卸载扩展会清除


## 开发说明
本项目为纯前端浏览器扩展，无需任何构建工具或依赖包。所有代码使用原生JavaScript实现，遵循Chrome扩展开发规范。
